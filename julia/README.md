# Julia

https://julialang.org/

## Setup

### Windows

https://julialang.org/downloads/ 에서 윈도 인스톨러를 제공합니다.

### macOS

[Homebrew](https://brew.sh/)로 설치할 수 있습니다.

```
$ brew install julia
```

### Linux

패키지 관리자로 설치할 수 있습니다. 아래는 APT의 예입니다.

```
# apt install julia
```

## Run

Julia는 네이티브 코드로 컴파일되지만 스크립트 언어처럼 실행할 수 있습니다.

```
julia 01.jl < input
```
