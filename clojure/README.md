## Setup

### Prerequisites

- java

### Install (Mac)
```bash
brew install clojure/tools/clojure
```
- 다른 OS 는 [여기](https://clojure.org/guides/getting_started) 를 참고하세요.

## Run
```bash
cd clojure
clj -M src/hello_santa.clj
```
